# sugarbeat

SugarBeat is a Python-based cake preorder management system built as my final project. The project was inspired by small home bakery businesses that usually manage orders manually through chats and notes.

The system allows customers to place fully customized cake orders by choosing flavors, sizes, layers, icing, colors, add-ons, and personalized messages. On the business side, the owner can manage preorder days and reminder schedules more efficiently.

The project was built using Object-Oriented Programming (OOP) concepts with inheritance between the Order and CakeOrder classes. It also uses Python’s threading module to automate daily reminders. I focused heavily on input validation, error handling, and wanting to be building something practical instead of just a basic academic project.
